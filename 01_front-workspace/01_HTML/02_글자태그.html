문단을 구분하는 태그 : p, pre

문단을 나누는 태그로 p태그와 pre태그가 있다. p태그는 태그 내에 줄바꿈 처리를 하고자 한다면 별도의 태그를 사용해야 한다.
** 줄바꿈 처리 **
그리고 공백은 한 개의 공백만 표시하므로 별도의 기호 문구를 사용해야 함
-> 기호 문구를 사용하지 않는 경우 : @ @
-> 기호 문구를 사용한 경우 : @     @

그 외 글자 관련 태그

일반적인 글자 표현 : 태그 없이 표현 가능

strong : 글자룰 굵게 표시하는 태그 (+스크린 리더)

em : 글자를 기울여서 표시하는 태그

mark : 형광펜 효과와 함께 표시하는 태그

u : 글자 하단에 밑줄을 표시하는 태그

s : 취소선을 표시하는 태그

글자를 작게 표시하는 태그 => small
