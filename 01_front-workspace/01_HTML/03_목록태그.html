목록 형태로 표시해주는 태그

  • 목록을 표시하는 태그 : li
  • 기본 값은 순서 없는 태그(ul)

    • ul : 순서 없는 목록 태그

    • HTML
    • CSS
    • JavaScript
    • jQuery
    • HTML
      • 글자 관련 태그 : p, pre
      • 목록 관련 태그 : ul, ol
    • CSS
    • JavaScript
    • jQuery

    ol : 순서 있는 목록 태그

    1. HTML
    2. CSS
    3. JavaScript
    4. jQuery

    Q1.숫자로 표시, 7부터 시작, 역순으로 목록 표시

    1. HTML
    2. CSS
    3. JavaScript
    4. jQuery

    Q2.영문으로 표시, 7번째 알파벳 소문자로 목록 표시

    1. HTML
    2. CSS
    3. JavaScript
    4. jQuery

    Q3.3번째 아라비아 숫자로 목록 표시

    1. HTML
    2. CSS
    3. JavaScript
    4. jQuery

    dl : 설명 관련 목록 태그 (주로 용어 정의 시 사용)

    제목(용어) 입력 태그
    설명 입력 태그
    프론트엔드 영역
    HTML, CSS, JavaScript 등의 언어를 사용하여 웹 브라우저 화면을 구성
    백엔드 영역
    Java, DB 등을 사용하여 사용자의 요청에 대한 처리 후 응답을 하여 서비스 제공
