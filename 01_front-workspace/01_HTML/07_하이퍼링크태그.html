하이퍼링크 태그

- 하이퍼링크 기능 : 웹 문서의 가장 핵심적인 기능
클릭을 통해 연결된 곳으로 이동시켜줌으로써 사용을 편리하게 하는 기능
텍스트를 클릭하거나, 이미지를 클릭하는 등의 방식으로 페이지 이동
=> 페이지 이동은 같은 페이지 내에서 이동하거나 다른 페이지로 이동할 수 있음

* href 속성

a 태그의 href 속성으로 링크를 만들 수 있음

  • 글자 관련 태그
  • 표 관련 태그
  • 네이버

* target 속성

연결할 페이지를 어떤 탭으로 연결할 것인지 설정하는 속성

  • _self : 기본값. 현재 탭에서 연결
    ex) 구글
  • _blank : 다른 탭으로 연결
    ex) 구글

* 이미지 태그(img) 이용해서 링크 연결

한 페이지 내에서 이동하는 링크 생성
