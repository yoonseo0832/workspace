jQuery

* jQuery란?

기존 자바스크립트 언어만으로 구현하기 복잡했던 구문들을 간소화하기 위해 만들어진 자바스크립트 기반의 라이브러리(유용한 기능 모음집)다.
즉, 자바스크립트 언어로 유용한 함수나 내용들을 다른 누군가가 정의해 놓은 것이다.
복잡한 자바스크립트 코드보다 짧고 쉽게 개발이 가능하다.

장점

  1. DOM 요소와 관련된 스크립트를 쉽게 구현할 수 있다.
  2. AJAX(비동기 통신 방법), 이벤트 처리 등의 기능을 폭넓게 지원해준다.
  3. jQuery와 관련된 확장된 플러그인, 오픈 API등을 지원해준다. (차트, 슬라이드, 캘린더, ...)

* jQuery 연결방법

head 태그 내에 script 태그로 외부 jQuery관련.js 파일을 연결해준다. (온라인/ 오프라인 방식)
- jQuery 공식 사이트 : jQuery 사이트

  1. 온라인 방식(CDN)방식 : 파일을 다운로드 하지 않고, 라이브러리를 제공하는 사이트 url을 연결하는 방식
  2. 오프라인 방식 : jQuery 라이브러리 파일을 직접 다운로드 받아 경로를 작성한다.
    • Compressed (배포전용) : .min.js
      주석, 들여쓰기, 띄어쓰기 등을 최소화하여 용량을 줄여 제공되는 파일이다.
    • UnCompressed (개발자전용) : .js
      개발자가 보기 쉽도록 주석, 코드 정렬이 되어있는 파일이다. (가독성 좋음)

* jQuery 구문 작성 방법

jQuery 구문으로 DOM요소를 다루는 구문을 사용하기 위해서 문서 상의 모든 요소들이 만들어진 이후 실행되어야 한다.
=> 순수 자바스크립트에서는 onload속성 내에서 작성해야함 

            [window.]onload = function() {
                // 화면이 모두 표시된 후 실행할 코드
            }

            * window.onload의 경우 해당 문서에서 단 한번만 작성 가능하다!

=> jQuery에서 onload 구문 (3가지) 
            (1) jQuery(document).ready(function() {
                // 실행할 코드
            });


            (2) $(document).ready(function() {
                // 실행할 코드
            });


            (3) $(function() {
                // 실행할 코드
            });

            * 제한없이 여러 번 사용 가능하다. 위에서부터 순서대로 실행이 된다.
